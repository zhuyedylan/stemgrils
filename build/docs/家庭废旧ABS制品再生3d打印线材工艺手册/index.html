家庭废旧ABS制品再生3D打印线材工艺手册

1. 材料来源与选择​

目标材料：丙烯腈-丁二烯-苯乙烯共聚物（ABS）。

常见来源：家庭中废弃的玩具、电器外壳、乐高积木等ABS塑料制品。（适量插图）

选择理由：熔融温度适中（210-240℃），热稳定性好，熔融时无有毒气体释放，且强度高，非常适合在家庭环境下通过小型设备进行熔融再生加工。

2. 回收与再生流程​

本流程旨在将废旧ABS制品转化为可用于FDM 3D打印机的线材（直径约1.75mm），包含以下五个核心阶段：

核心任务

关键产出

阶段一：前期处理​

1. 收集清洁后的ABS废品。
2. 使用小型粉碎机将其粉碎为粒径约3-4mm的颗粒。
3. 将颗粒置于100℃的烘干设备中干燥4小时，去除水分。干燥的ABS再生颗粒

阶段二：熔融挤出​

1. 将干燥后的ABS颗粒加入单螺杆挤出机。
2. 设定并加热至目标温度（参见第3部分工艺参数）。
3. 启动挤出机，均匀进料，通过模头挤出熔融的塑料线材。初步成型的塑性线材

阶段三：冷却定型​

1. 挤出的高温线材立即通过风冷（或水冷槽）进行冷却固化。
2. 使用牵引机以恒定速度牵引线材，确保直径均匀。固化的ABS线材

阶段四：卷绕与检测​

1. 将冷却后的线材卷绕成盘。
2. 使用游标卡尺在多个点位测量线材直径，确保其稳定在1.75mm±0.05mm范围内。成品线材卷

阶段五：3D打印验证​

1. 将再生线材装入3D打印机，使用推荐的打印参数（参见第3部分）进行测试打印。
2. 首先打印一个小测试块（如1cm x 1cm），验证线材的打印流畅性和成型质量。3D打印成品

3. 关键工艺参数​

以下参数基于实验验证，可根据具体设备微调：

预处理参数：

烘干温度：100℃

烘干时间：4小时

熔融挤出参数：

挤出机熔融段温度：205℃

螺杆转速：2.9 r/min​ （对应实验中的牵引速度）

牵引速度：2.9 mm/s

冷却方式：风冷

线材质量标准：

目标直径：1.75 mm

允许公差：±0.05 mm​ （即1.70mm ~ 1.80mm为合格）

外观：表面光滑均匀，无气泡、结节或竹节现象。

3D打印参数建议：

打印喷嘴温度：250℃

热床温度：60℃

（可选）层厚：0.2mm；填充率：20%；打印速度：40-50mm/s

4. 安全提示与注意事项​

个人防护：操作过程中必须佩戴口罩及耐热手套，防止吸入塑料受热可能产生的微量气体，并避免烫伤。

设备安全：确保挤出机、3D打印机等电气设备接地良好，远离易燃物。加热部件高温，切勿用手直接触碰。

材料安全：仅限使用明确标识为ABS的塑料。严禁处理PVC等高温下会释放有毒气体的塑料。

通风：整个加工过程应在通风良好的环境中进行。

参数监控：密切关注挤出温度，避免温度过高导致材料降解或设备故障（如文档中PET实验因温度过高损坏设备）。

5. 总结​

本工艺验证了利用家庭常见的单螺杆挤出设备，将废旧ABS制品再生为合格3D打印线材的可行性。通过控制合理的预热、温度、牵引速度及冷却条件，可以获得直径均匀、表面光滑的线材，并成功用于3D打印。这为在家庭和学校等场景下实现塑料的闭环循环提供了一套安全、可行的实践方案。
