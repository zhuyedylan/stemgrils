# 社区家庭或校园废旧高分子材料再生3D打印项目

## 项目背景

本项目是"探知未来科技女性培养计划"的重要组成部分，旨在通过科技创新培养初中生的实践能力和环保意识。

### 什么是废旧高分子材料再生3D打印？

随着经济发展和生活水平提高，日常生活中产生的塑料废弃物越来越多。这些废弃物如果得不到有效处理，会对环境造成严重污染。

**废旧高分子材料再生3D打印**是一种将废弃塑料（如PLA塑料杯、咖啡渣等）通过科学方法回收再利用，转化为3D打印线材的技术。这种技术不仅减少了环境污染，还培养了青少年的科技创新能力。

## 项目目标

1. **环保教育**：让青少年了解塑料污染的危害和资源循环利用的重要性
2. **科技创新**：培养学生的科学思维和动手实践能力
3. **公益实践**：参与有社会意义的科技创新活动

## 项目内容

### 主要活动

- 废旧PLA塑料回收与分类
- 咖啡渣与PLA混合材料实验
- 3D打印线材制作工艺学习
- 创意作品设计与打印

### 产出成果

学生通过项目实践，可以制作出各种有趣的3D打印作品，如：
- 咖啡渣再生材料制作的杯垫
- 微型花盆
- 创意摆件
