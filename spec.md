## 1. Project Overview
Platform for junior high students to document 3D printing process manuals.

## 2. Technical Stack
- **Framework:** Docusaurus 3.x.
- **Core:** React 18+, MDX.
- **Automation:** Node.js + mammoth.js.

## 3. Key Features
- Automated Word-to-MD Workflow.
- Interactive "Golden Ratio" calculator component.
- Hierarchical manual navigation.

## 4. Content Architecture
- **Home:** Green Tech concepts.
- **Docs/Process:** Sorting, Sieving, Extrusion.
- **Docs/Safety:** High-visibility protocols.
